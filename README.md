# Практическая работа №23: Аутентификация и авторизация

## Описание проекта
Полнофункциональная система аутентификации и авторизации с использованием JWT токенов, хешированием паролей и ролевой моделью доступа.

## Технологии
- Node.js + Express.js
- SQLite3 (база данных)
- JWT (JSON Web Tokens)
- bcryptjs (хеширование паролей)
