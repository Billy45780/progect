Тестовый клиент для Node.js API

Интерфейс для тестирования REST API сервера

Результат:

Статус: -
Время: - мс

        


        

            
Примеры использования:

            

                GET /api/users - получить всех пользователей
            

            

                POST /api/products - добавить новый товар
            

            

                PUT /api/products/1 - обновить товар
            

            

                DELETE /api/products/1 - удалить товар
